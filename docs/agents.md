## Lock-and-Key Addressing

**CRITICAL:** Every message MUST include agent identity to prevent role drift and hallucinated coordination.

**Format:**
```
Δ 👾 ∇ Δ [EMOJI] [Agent Name]: [your query]
```

**Example:**
```
Δ 👾 ∇ Δ 🐋 DeepSeek: Prove the Riemann hypothesis
```
